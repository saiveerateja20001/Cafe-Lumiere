# Café Lumière v2.0 - Color Palette Update

## New Color Scheme: Modern Green & Pink

The v2.0 version features a completely refreshed, modern color palette with soft greens and vibrant pinks.

### Color Variables

```css
--primary: #2D6A4F       /* Deep Green - Main accent color */
--secondary: #D8F3DC     /* Light Green - Secondary backgrounds */
--accent: #40916C        /* Medium Green - Hover states */
--light: #EDF5F0         /* Very Light Green - General backgrounds */
--white: #FFFFFF         /* White - Cards and content areas */
--dark: #1B3A2A          /* Dark Green - Text color */
--success: #2D6A4F       /* Deep Green - Success indicators */
--warning: #F876C5       /* Pink - Warnings and alerts */
--ordered: #40916C       /* Medium Green - Ordered status */
--preparing: #F876C5     /* Pink - Preparing status */
--ready: #52B788         /* Mint Green - Ready status */
--served: #A8D8D8        /* Teal - Served status */
```

### Visual Changes

#### Before (v1.0)
- **Theme**: Warm brown/beige (coffee-inspired)
- **Primary**: #6B4226 (Brown)
- **Secondary**: #D4A574 (Tan)
- **Accent**: #8B6F47 (Olive)
- **Background**: Cream/Beige gradients

#### After (v2.0)
- **Theme**: Fresh green with pink accents (modern, vibrant)
- **Primary**: #2D6A4F (Deep Forest Green)
- **Secondary**: #D8F3DC (Soft Mint Green)
- **Accent**: #40916C (Garden Green)
- **Background**: Light green gradients

### Updated Elements

✅ **Backgrounds**
- Body gradient: Green tones (#EDF5F0 → #D8F3DC)
- Menu cards: Fresh green gradients
- Cart section: Soft green styling
- Kitchen board: Light green backgrounds
- Display board: Mint green sections

✅ **Interactive Elements**
- Buttons: Deep green primary with pink warnings
- Hover states: Medium green (#40916C)
- Remove buttons: Vibrant pink (#E85D9F)
- Card borders: Light green (#D8F3DC)

✅ **Status Indicators**
- Ordered: Medium Green (#40916C)
- Preparing: Vibrant Pink (#F876C5)
- Ready: Fresh Mint (#52B788)
- Served: Cool Teal (#A8D8D8)

✅ **Text & Typography**
- Primary text: Dark Green (#1B3A2A)
- Headers: Deep Green (#2D6A4F)
- Secondary text: Medium Green (#40916C)

### Color Psychology

- **Green**: Represents freshness, calm, growth, and the café's healthy beverages
- **Pink**: Adds vibrancy, energy, and modern appeal
- **Mint/Teal**: Creates visual interest and differentiates status states

### Files Modified

- `frontend/static/style.css` - Complete color palette overhaul
- All gradients updated to green/pink tones
- Scrollbar colors updated
- Animation pulses reflect new palette

### Browser Compatibility

The new color scheme uses standard CSS color values and is compatible with all modern browsers.
