# Café Lumière v3.0 - Color Palette

## Modern Teal & Coral/Orange Palette

The v3.0 version features a sophisticated and contemporary design with Teal and Coral/Orange colors.

### Color Variables

```css
--primary: #0B7285       /* Deep Teal - Main accent color */
--secondary: #C3FAE8     /* Light Cyan - Secondary backgrounds */
--accent: #15AABF        /* Bright Teal - Hover states */
--light: #E7F5FA         /* Very Light Teal - General backgrounds */
--white: #FFFFFF         /* White - Cards and content areas */
--dark: #0B2C3D          /* Dark Teal-Blue - Text color */
--success: #0B7285       /* Deep Teal - Success indicators */
--warning: #FF6B6B       /* Coral Red - Warnings and alerts */
--ordered: #15AABF       /* Bright Teal - Ordered status */
--preparing: #FF922B     /* Warm Orange - Preparing status */
--ready: #FFA94D         /* Light Orange - Ready status */
--served: #74C0FC        /* Sky Blue - Served status */
```

### Visual Element Updates

✅ **Backgrounds**
- Body gradient: Teal tones (#E7F5FA → #C3FAE8)
- Menu cards: Light cyan gradients
- Cart section: Teal styling
- Kitchen board: Light cyan backgrounds
- Display board: Cyan teal sections

✅ **Interactive Elements**
- Buttons: Deep teal primary with coral warnings
- Hover states: Bright teal (#15AABF)
- Remove buttons: Vibrant coral red (#FF6B6B)
- Card borders: Light cyan (#C3FAE8)

✅ **Status Indicators**
- Ordered: Bright Teal (#15AABF)
- Preparing: Warm Orange (#FF922B)
- Ready: Light Orange (#FFA94D)
- Served: Sky Blue (#74C0FC)

### Color Psychology

- **Teal**: Represents calmness, professionalism, trust, and sophistication
- **Coral/Orange**: Adds warmth, energy, and brings attention to important actions
- **Sky Blue**: Creates visual interest and represents completion/success
